
# Job Assignment Pay Type

Enumerates the possible pay types that a job can be assigned.

## Enumeration

`Job Assignment Pay Type`

## Fields

| Name | Description |
|  --- | --- |
| `NONE` | The job does not have a defined pay type. |
| `HOURLY` | The job pays an hourly rate. |
| `SALARY` | The job pays an annual salary. |
