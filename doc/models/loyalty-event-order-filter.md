## Loyalty Event Order Filter

Filter events by the order associated with the event.

### Structure

`LoyaltyEventOrderFilter`

### Fields

| Name | Type | Description |
|  --- | --- | --- |
| `order_id` | `string` | The ID of the [order](#type-Order) associated with the event. |

### Example (as JSON)

```json
{
  "order_id": "order_id6"
}
```
